# JPGE
JohnÂ´s Project Game Engine (JPGE) is tiny software based Game Engine writen in Java. It is designed for high performance and based only on the default Java graphics libraries (AWT, SWING...).

# Features
* Pure Java
* Fixed Point only
* Shaders
* No matrices
* Gouraud shading
* Affine Texture mapping with gouraud shading
